# IBM Data Science Capstone Project

This repository contains the final Capstone Project for the **IBM Data Science Professional Certificate**, applying machine learning algorithms, data science methodologies, and data visualization to solve a real-world predictive modeling problem.

---
📋 Project Overview
The main objective of this project was to tackle a complex business or operational challenge by leveraging historical data, building predictive models, and evaluating their performance to provide data-driven recommendations.

🛠️ Technologies and Tools
* **Programming Language:** Python (Pandas, NumPy, Scikit-learn)
* **Machine Learning & Modeling:** Classification / Regression algorithms (Logistic Regression, SVM, Decision Trees, KNN)
* **Data Visualization:** Matplotlib, Seaborn, Folium (interactive maps)
* **Data APIs & Collection:** REST APIs, Web Scraping (BeautifulSoup)
* **Databases:** SQL for data querying and wrangling
* **Environment:** Jupyter Notebooks

---

## 🔄 Project Phases (Data Science Methodology)

1. **Business Understanding & Data Collection:** 
   * Defining the objective and gathering raw data via web scraping, APIs, or SQL databases.
2. **Data Preparation & Wrangling:**
   * Handling missing values, encoding categorical variables, feature engineering, and normalization.
3. **Exploratory Data Analysis (EDA) & Visualization:**
   * Using statistical analysis, correlation matrices, and spatial mapping (Folium) to understand feature importance and data distribution.
4. **Machine Learning Model Development:**
   * Training multiple classification or regression models, tuning hyperparameters, and evaluating performance using metrics like Accuracy, F1-Score, Precision, Recall, and ROC-AUC.
5. **Model Evaluation & Conclusion:**
   * Comparing model results to determine the best-performing algorithm and translating findings into strategic recommendations.
